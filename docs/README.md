## Archive

Le dossier `docs/archive/` contient les anciennes notes de phase, passes de correction, sprints et documents de travail.

Ces documents sont conservés pour historique, mais ne doivent plus être lus comme source principale.

## Règle de maintenance

Avant de créer un nouveau document, vérifier si le contenu peut s'intégrer dans l'un des 7 documents principaux. Objectif : éviter la multiplication des fichiers concurrents.
