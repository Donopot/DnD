## Archive

Le dossier `docs/archive/` contient les anciennes notes de phase, passes de correction et documents de sprint.

Ces documents sont conservés pour historique, mais ne doivent plus être lus comme source principale.

## Règle de maintenance

Avant de créer un nouveau document, vérifier si le contenu peut aller dans :

- `project-roadmap.md`
- `backend-roadmap.md`
- `frontend-panels.md`
- `plan-refonte-map.md`
- `deployment.md`

Objectif : éviter la multiplication des fichiers concurrents.
