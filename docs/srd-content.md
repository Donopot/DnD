# SRD — Règles de base D&D 5e

## Conditions

| Condition | Effet |
|-----------|-------|
| **À terre** | Désavantage aux attaques. Les attaques à 5ft ont avantage, les attaques à distance désavantage. |
| **Aveuglé** | Échec automatique jets basés sur la vue. Désavantage aux attaques. Avantage aux attaquants. |
| **Charmé** | Ne peut attaquer le charmeur. Avantage du charmeur aux interactions sociales. |
| **Assourdi** | Échec automatique jets basés sur l'ouïe. |
| **Empoisonné** | Désavantage aux jets d'attaque et de caractéristique. |
| **Étourdi** | Actions et réactions impossibles. Désavantage DEX. Avantage aux attaquants. |
| **Inconscient** | Inconscient, à terre. Échec automatique DEX/STR. Avantage aux attaquants. Attaques à 5ft = critiques. |
| **Neutralisé** | Pas d'actions ni réactions. |
| **Paralysé** | Neutralisé + échec auto DEX/STR + critiques à 5ft. |
| **Pétrifié** | Transformé en pierre. Résistance à tous les dégâts. Immunité poison/maladie. |
| **Terrifié** | Désavantage aux jets + ne peut s'approcher de la source. |
| **Agrippé** | Vitesse = 0. Libération sur test Athlétisme/Acrobaties DD fixe. |
| **Restreint** | Vitesse = 0. Désavantage aux attaques + DEX. Avantage aux attaquants. |
| **Fatigué** (niv 1-5) | Niv 1: désavantage capacités | Niv 2: vitesse ÷2 | Niv 3: désavantage attaques/sauvegardes | Niv 4: PV max ÷2 | Niv 5: vitesse 0 | Niv 6: mort |

## Actions en combat

| Action | Description |
|--------|-------------|
| **Attaquer** | Une attaque d'arme ou à mains nues. Extra Attack au niv. 5+ pour certaines classes. |
| **Lancer un sort** | Un sort avec temps d'incantation de 1 action. Pas 2 sorts niveau 1+ par tour (sauf cantrip + action bonus). |
| **Foncer** (Dash) | Double la vitesse de déplacement pour ce tour. |
| **Esquiver** (Dodge) | Les attaquants ont désavantage. Avantage aux jets de DEX. |
| **Se cacher** (Hide) | Test de Discrétion contre Perception passive des ennemis. |
| **Aider** (Help) | Avantage au prochain jet de capacité d'un allié, ou distraire un ennemi pour avantage d'attaque. |
| **Se désengager** (Disengage) | Pas d'attaques d'opportunité ce tour. |
| **Prêt** (Ready) | Prépare une action à déclencher selon une condition. Consomme la réaction. |
| **Utiliser un objet** | Potion, levier, torche, etc. |
| **Chercher** (Search) | Test de Perception ou Investigation. |

## Jets de sauvegarde

Chaque classe maîtrise 2 sauvegardes :
- **Barbare** : FOR, CON
- **Barde** : DEX, CHA
- **Clerc** : SAG, CHA
- **Druide** : INT, SAG
- **Ensorceleur** : CON, CHA
- **Guerrier** : FOR, CON
- **Magicien** : INT, SAG
- **Moine** : FOR, DEX
- **Occultiste** : SAG, CHA
- **Paladin** : SAG, CHA
- **Rôdeur** : FOR, DEX
- **Roublard** : DEX, INT

DD de sauvegarde = 8 + bonus de maîtrise + modificateur de caractéristique

## État de santé

- **PV > 0** : conscient, normal
- **PV = 0** : inconscient, jets de mort (3 succès = stable, 3 échecs = mort)
- **PV = -max** : mort instantanée
- **Stable** : 0 PV, inconscient. Reprend 1 PV après 1d4 heures.

Jets de mort : d20. 1 = 2 échecs. 20 = regagne 1 PV. 2-9 = 1 échec. 10-19 = 1 succès.

## Repos

- **Repos court** (1h) : dépenser dés de vie pour récupérer des PV. Récupération de certaines capacités.
- **Repos long** (8h) : PV max restaurés. Dés de vie ÷2 restaurés. 1 seul par 24h.

## Attaques d'opportunité

Quand une créature hostile quitte volontairement ta portée d'allonge (5 ft par défaut), tu peux utiliser ta réaction pour faire une attaque d'opportunité (une seule attaque de mêlée). Se désengager annule cette possibilité.

## Avantage / Désavantage

- **Avantage** : lancer 2d20, garder le meilleur. Équivalent à ~+3.3 en moyenne.
- **Désavantage** : lancer 2d20, garder le pire. Équivalent à ~-3.3 en moyenne.
- Ne se cumulent pas (un avantage annule tout désavantage et vice versa).

## Couverture

- **½ couverture** : +2 CA et jets de DEX
- **¾ couverture** : +5 CA et jets de DEX
- **Totale** : ne peut pas être ciblé directement

## Visibilité et lumière

- **Lumière vive** : vision normale
- **Lumière faible** : désavantage Perception (vision)
- **Obscurité** : aveuglé pour les créatures sans vision dans le noir
- **Vision dans le noir** : voit en obscurité comme lumière faible (nuances de gris, pas de couleurs)
